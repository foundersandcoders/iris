TUI Redesign Roadmap

iris · docs/roadmaps/tui-redesign.md · prefix TR

Take the Iris TUI from “functional” to a top-class, lazygit-class terminal application — framed panels, persistent chrome, a coherent interaction grammar, and a legible semantic colour system. Phases are dependency-ordered: foundations first, because the shell rollout and signature features build on them.
