# {{PROJECT_NAME}} Documentation

This directory contains all project documentation.

## Structure

### `/roadmaps/`
Project roadmaps tracking milestones, tasks, and progress.

- `mvp.md` - MVP development roadmap
- Future releases will have additional roadmap files

### `/work-records/`
Development journal documenting daily/weekly progress, decisions, and learnings.

Format: `YYYY-MM-description.md`

### `/adrs/`
Architecture Decision Records documenting significant technical decisions.

Format: `NNN-title-of-decision.md`

### `/technical/`
Technical documentation including:
- API documentation
- Component documentation
- System architecture
- Database schemas
- Integration guides

## Contributing

When adding documentation:
1. Use the appropriate directory for the document type
2. Follow existing naming conventions
3. Link between documents where relevant
4. Keep documents focused and concise

## Portfolio Evidence

Development work documented here automatically contributes to portfolio evidence through git hooks.
